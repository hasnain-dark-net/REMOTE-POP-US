# Local Popup Demo — HasnainDarkNet

**One-line:** Simple local Windows popup demo for educational testing (VM-only).  
**⚠️ Important:** Run this **only** on your own machine or inside an isolated VM. Do **not** run on other people's systems or on public networks.

---

## What it does
A tiny demo that shows how to display a popup message on the same Windows machine using `msg` (or a PowerShell GUI fallback). No remote access, persistence, or obfuscation included.

---

## Quick Start
1. Put `local_popup_demo.bat` (or the provided PowerShell fallback) in a folder.  
2. Double-click `local_popup_demo.bat` or run it from Command Prompt:
```bat
local_popup_demo.bat
